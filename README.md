# HumanityFirst Global Relief

A professional, responsive humanitarian donation platform prototype built with Next.js and Tailwind CSS.

## Features
- Multi-category support (Food, Education, Healthcare, Disaster Relief, Environment, Women & Child Welfare, Animal Welfare)
- Project listings with detailed transparency (budgets, invoices, donors, updates)
- Donor dashboard with charts, analytics, and placeholders for map/2FA
- Donors wall, transparency dashboard, and contact page
- Dummy JSON data for projects, donors, transactions, updates, and invoices
- Security placeholders for HTTPS/SSL, admin login, audit logs, and 2FA UI

## Getting started
1. Install dependencies
```bash
npm install
```
2. Run the development server
```bash
npm run dev
```
3. Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment
Deploy easily to Vercel or Netlify. Ensure environment is served over HTTPS and configure authentication for admin/donor areas when backend is connected.
